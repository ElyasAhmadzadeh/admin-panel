# Admin Panel - پنل مدیریت حرفه‌ای 

یک پنل مدیریت مدرن، زیبا و ریسپانسیو با طراحی راست‌به‌چپ (RTL)، مناسب برای داشبورد ادمین وبسایت‌ها، فروشگاه‌ها یا سیستم‌های مدیریتی.

## پیش‌نمایش زنده
[پیش‌نمایش زنده](https://your-username.github.io/admin-panel) *(به‌زودی)*

## ویژگی‌های پروژه

- طراحی کاملاً ریسپانسیو و مناسب برای تمام دستگاه‌ها (دسکتاپ، تبلت، موبایل)
- پشتیبانی کامل از جهت **RTL** (راست‌به‌چپ) برای زبان فارسی
- سایدبار ثابت با تصویر پس‌زمینه و افکت گرادیانت تیره
- منوی سایدبار با حالت فعال (active) و افکت hover جذاب
- نوار بالا (Top Navbar) با جستجو، اعلان‌ها، پیام‌ها و مودال خروج
- کارت‌های وضعیت (Status Cards) با افکت hover و سایه
- لیست وظایف (Tasks) با چک‌باکس و تغییر رنگ border هنگام تکمیل
- جدول‌های مرتب و زیبا با pagination
- آکاردئون (Accordion) برای نمایش تاریخچه فعالیت‌ها و نظرات
- بخش پست جدید با آیکون‌های سریع و نمایش تاریخ/ساعت
- فونت سفارشی **IranYekan** برای ظاهر حرفه‌ای پارسی
- انیمیشن‌ها و transitionهای نرم در سراسر پنل
- جاوااسکریپت خالص برای مدیریت حالت فعال منو و چک‌باکس وظایف

## تکنولوژی‌های استفاده‌شده

- HTML5
- CSS3 
- JavaScript (vanilla JS)
- Bootstrap 4.5.3
- Font Awesome 7
